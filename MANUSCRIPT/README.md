# MANUSCRIPT

Manoscritto scientifico in formato R Markdown (.Rmd).

## Formato consigliato
- Utilizzare R Markdown per integrare codice R, testo e referenze
- File `.bib` per la bibliografia
- Template per riviste specifiche se necessario

## Esportazione
- PDF via `knitr` e LaTeX
- Word via Pandoc se richiesto dalla rivista
